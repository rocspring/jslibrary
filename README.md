jslibrary
=========

##一些常用的js组件库

##common(js的基础类库)

*classOperate.js  ：一个class操作库

*findJqObjEvents.js : 查找引用jquery后，dom对象上注册的事件

*jsonp.js : jsonp的原生实现

*nativeAjax.js : ajax的原生实现


##lib(js的常用组件)

*chaptersplit.js : 把章节内容按屏幕进行分页

*exposureStatis.js : 页面元素的曝光统计

*orientationChange.js : 屏幕旋转的事件封装
